# M2 (Discovery + Order)

## Scope
Discovery, Menu, Cart, Order

## APIs
GET /restaurants
GET /restaurants/{id}

GET /menu/{restaurant_id}

POST /cart/add
POST /cart/remove
GET /cart

POST /orders
GET /orders/{id}

## Entities
Restaurants(id, name, location, status)
MenuItems(id, restaurant_id, price, availability)
Cart(user_id, items)
Orders(id, user_id, restaurant_id, status, total)

## Rules
- One cart per user
- One restaurant per cart
- Validate item availability before order
- Order only after validation

## Output
Customer:
- Browse restaurants
- View menu
- Add to cart
- Place order

Restaurant:
- Accept/reject order
- Update status (Accepted → Preparing → Ready)
