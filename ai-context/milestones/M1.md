# M1 (Foundation)

## Scope
Auth, Profiles, RBAC, Base System

## Actors
Customer, Driver, Restaurant, Admin

## APIs
POST /auth/otp
POST /auth/verify
POST /auth/logout

GET /profile
PUT /profile

## Entities
Users(id, role, mobile, status)
CustomerProfile(user_id, name, addresses)
DriverProfile(user_id, vehicle, docs, status)
Restaurant(user_id, business_details, status)

## Rules
- OTP login mandatory
- Role-based access control required
- Admin approval required (driver, restaurant)
- Single user → single role

## Output
- All users can register/login
- Profiles created
- Admin can approve/reject users
- Base auth system ready
