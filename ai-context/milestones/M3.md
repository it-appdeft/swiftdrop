# M3 (Payment + Admin)

## Scope
Payments, Admin Control, Offers

## APIs
POST /payments/initiate
POST /payments/verify

GET /admin/orders
GET /admin/users

POST /offers/apply

## Entities
Payments(order_id, status, amount, txn_id)
Offers(id, type, value, rules)
Config(key, value)

## Rules
- Payment success → order created
- Payment failure → no order
- Prevent duplicate payments
- Coupon validation mandatory

## Output
- Payment flow working
- Admin dashboard (basic)
- Offers & coupons working
