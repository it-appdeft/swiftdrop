# M4 (Delivery + Tracking)

## Scope
Driver, Assignment, Tracking, Completion

## APIs
POST /driver/online
POST /driver/accept

POST /driver/location
GET /tracking/{order_id}

POST /orders/status

## Entities
Drivers(id, status, location)
Assignments(order_id, driver_id, status)

## Rules
- Assign after restaurant ready
- Max 3 driver attempts
- Fallback to admin
- Location update every 5 sec

## Output
- Driver assignment working
- Delivery lifecycle complete
- Real-time tracking enabled
- Order completion flow
